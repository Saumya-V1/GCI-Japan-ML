# GCI University of Japan – Machine Learning Course 2026

This repository contains projects completed as part of the **Machine Learning Course 2026** at **GCI (Global Consumer Intelligence), University of Tokyo, Japan**.

Throughout the course, I applied machine learning techniques to real-world datasets, covering the complete data science workflow:

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Feature engineering
* Machine learning model development
* Model evaluation and validation
* Business insight generation and recommendations

## Projects

### NFL Draft Prediction

Developed predictive models to estimate the likelihood of college football players being selected in the NFL Draft using athletic, positional, and performance metrics.

### Telecom Customer Churn Prediction

Built an XGBoost-based churn prediction system to identify at-risk telecom customers and support proactive retention strategies through data-driven decision making.
